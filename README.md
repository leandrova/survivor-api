Survifor FC
=============

## Settings

Create `.env` file 

* [DATA_HOST]
* [DATA_USER]
* [DATA_PASS]
* [DATA_BASE]
* [SERVER_PORT]

## Prerequisites
* [Npm]
* [Gulp]
* [Node]

## Setup

  - Run `npm install`
  - Run `gulp build`

## Running

  - Run `Em /app -> node app.js`

## Building

`gulp build`
